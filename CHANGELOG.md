## 0.1.0

- Initial release.
- Plays a sound on Claude Code permission requests, idle prompts, and elicitation dialogs.
- Optional sound on Stop (task finished).
- Auto-installs/removes Claude Code hooks into settings.json.
- Cross-platform native audio playback (macOS / Windows / Linux / WSL).
- Configurable sounds, workspace scoping, toast notifications, and custom player command.
